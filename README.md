# 🛍️ Trendify

**Trendify** is an **E-commerce Web Application** built with **Angular**.  
It includes core features like product listing, product details, shopping cart, login/signup with Reactive Forms, and extra sections like About & Contact.  

---

## 🚀 Features
- 🛒 Display products with a clean UI.
- 🔍 Search products by name.
- 🗂️ Filter products by category (Electronics, Clothes, Home Appliances).
- 📄 View detailed product information.
- ➕ Add products to cart.
- 💳 Checkout page.
- 🔑 **Login / Signup using Reactive Forms** with built-in validation.
- ℹ️ Extra pages: About, Contact.
- 🎨 Unified purple theme for buttons and highlights.
- 🕒 Clock component that updates in real time and stops when hidden.
- 📌 Navbar with active links using **RouterLinkActive**.

---

## 🛠️ Tech Stack
- **Angular 17+**
- **Bootstrap 5**
- **TypeScript**
- **Reactive Forms**
- **HTML & CSS**
